Controlling floats

\FloatBarrier from placeins

Using the placeins package to force floats (figures and tables) to only appear inside the (sub)sections where they are defined:

  % Force floats inside respective (sub)sections.
  \usepackage[section]{placeins}
  \let\oldsubsection\subsection
  \renewcommand\subsection{\FloatBarrier\oldsubsection}

\clearpage

The command \clearpage will not only start a new page, but will also force any unset floats to be set before the page break. Read more about \clearpage.
