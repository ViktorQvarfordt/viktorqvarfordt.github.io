Automated SSH access, no password

Sometimes one needs to have an automated process connect to a remote host via ssh. For this purpose I like to have a special key with no passphrase, normally passphrases should be used.

Create the passphrase-less private and public key:

$ ssh-keygen -t rsa -f ~/.ssh/id_rsa_nopass

Put the public key on remote:

$ ssh-copy-id -i ~/.ssh/id_rsa_nopass.pub '-p 22 user@remote-host'

More info on this ssh keys:

Configuring SSH client

See man ssh_config.

$ cat ~/.ssh/config
Host alias
    HostName example.com
    User user
    Port 44
    IdentityFile ~/.ssh/id_rsa_nopass

Instead of doing ssh user@example.com:44 -i ~/.ssh/id_rsa_nopass we can now simply do ssh alias.
